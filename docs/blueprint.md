# **App Name**: MediCare – AI Health Companion

## Core Features:

- Prescription Upload & Smart Scan: Enable users to scan prescriptions with an intelligent camera featuring auto-capture and blur detection, or upload image/PDF files for analysis.
- Instant AI Report Analysis: Leveraging Grok AI, this feature processes scanned/uploaded prescriptions to instantly generate a one-line summary, plain-language diagnosis, medication schedule (table format), potential side effects, and a follow-up checklist.
- Language Toggle: Allow users to instantly switch the displayed AI analysis results between English and Hindi for improved comprehension.
- Emergency Help Mode: Powered by Grok AI, this critical emergency interface provides immediate actions, pre-written advice (e.g., 'Call emergency services'), and voice input/output capabilities for urgent situations, acting as a 'tool' for rapid response.
- Context-Aware AI Assistant: A floating, animated Grok AI assistant 'tool' that leverages the currently analyzed prescription data to answer user questions via quick buttons, voice input, and voice output, without providing external medical advice.
- Digital Health Locker: Enable users to save analyzed prescription reports to a personal, secure health locker for future reference, with a timeline or card grid view.
- AI Processing Visual Feedback: Display an engaging, animated sequence to visualize the AI's progress during prescription processing (e.g., 'Reading prescription...', 'Understanding medical terms...', 'Structuring data...').

## Style Guidelines:

- Primary color: A professional, clear blue (#0066FF) representing trust and healthcare, used for primary actions and key elements.
- Background color: A very light, desaturated blue-gray (#EBF0F2) to evoke a clean, calm, and spacious 'hospital-grade' UI.
- Accent color: A fresh, bright aqua (#66E0E0) to highlight secondary actions or indicate positive outcomes, offering a distinct contrast to the primary blue.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring high readability for medical information.
- Utilize minimalist, line-based, and friendly vector icons, aligning with the clean, modern 'hospital-grade' UI. Icons should be clear and instantly recognizable, especially for medical contexts.
- Adopt a clean, card-based layout with rounded corners and subtle shadows for clear information hierarchy. Initial views prioritize immediate action (scan/upload), while post-analysis includes a sidebar for additional features.
- Incorporate smooth, purposeful animations (e.g., using Framer Motion) for UI transitions, content reveals, AI processing feedback, and interactive elements like the floating AI assistant, prioritizing speed and clarity over excessive flair.
